Nebuleuse is currently still in Alpha, important features are not yet implemented and bugs will be present

Working Features

  • Users informations
  • Achievements
  • Statistics
  • Administration dashboard

Planned Features

These features are not yet under development but are the next steps in the evolution of Nebuleuse.

  • Update system
  • Server browser
  • Lobby based matchmaking
  • Items and inventory

Clients

Currently only a C++ client is available. This client is compatible with Unreal Engine 4.
C# and lua clients are planned in the near future but contributers are welcome to start working on one for your favorite language.
