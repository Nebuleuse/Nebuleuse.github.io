Nebuleuse is currently still in Alpha, most features are not yet implemented and bugs will be present

Features

  • Users informations
  • Achievements
  • Statistics
  • Administration panel

Planned Features

These features are not yet under developpement but are the next steps in the evolution of Nebuleuse.

  • Updates
  • Server browser
  • Lobby based matchmaking
  • Items and inventory

Clients

Currently only a C++ client is available. C# and lua support as planned in the near future but contributers are welcome to start working on one for your favorite language.
